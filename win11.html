Microsoft Activation Scripts (MAS)
Microsoft 激活脚本 (MAS)

Open-source Windows and Office activator featuring HWID, Ohook, TSforge, KMS38, and Online KMS activation methods, along with advanced troubleshooting.
开源 Windows 和 Office 激活器，具有 HWID、Ohook、TSforge、KMS38 和在线 KMS 激活方法以及高级故障排除功能。

How to Activate Windows / Office / Extended Updates (ESU)?
如何激活 Windows / Office / 扩展更新 (ESU)？​

Method 1 - PowerShell ❤️
方法 1 - PowerShell❤️​

info  信息

  1. Open PowerShell  打开 PowerShell
    Click the Start Menu, type PowerShell, then open it.
    单击 “开始菜单” ，键入 PowerShell ，然后打开它。

  2. Copy and paste the code below, then press enter.
    复制并粘贴下面的代码，然后按回车键。

    • For Windows 8, 10, 11: 📌
      对于 Windows 8、10、11 ：📌       
      irm https://get.activated.win | iex
      If the above is blocked (by ISP/DNS), try this (needs updated Windows 10 or 11):
      如果上述操作被阻止（被 ISP/DNS），请尝试以下操作（需要更新 Windows 10 或 11）：       
      iex (curl.exe -s --doh-url https://1.1.1.1/dns-query https://get.activated.win | Out-String)
    • For Windows 7 and later:
      对于 Windows 7 及更高版本：       
      iex ((New-Object Net.WebClient).DownloadString('https://get.activated.win'))
    • Script not launching❓Use Method 2.
      脚本未启动❓使用方法 2 。

  3. The activation menu will appear. Choose the green-highlighted options to activate Windows or Office.
    激活菜单将会出现。 选择绿色突出显示的选项来激活 Windows 或 Office。

  4. Done!  完毕！

Method 2 - Traditional (Windows Vista and later)
方法 2 - 传统（Windows Vista 及更高版本）​

info  信息
  1. Download the script: MAS_AIO.cmd or the full ZIP.
    下载脚本： MAS_AIO.cmd 或完整的 ZIP 。
  2. Run the file named MAS_AIO.cmd.
    运行名为 MAS_AIO.cmd 的文件。
  3. You will see the activation options. Follow the on-screen instructions.
    您将看到激活选项。请按照屏幕上的说明进行操作。
  4. That's all.  就这样。
tip  提示
  • Some ISPs/DNS block access to our domains. You can bypass this by enabling DNS-over-HTTPS (DoH) in your browser.
    某些 ISP/DNS 会阻止访问我们的域名。您可以通过在浏览器中启用 DNS-over-HTTPS (DoH) 来绕过此限制。
  • Having trouble❓Connect with us here.
    遇到问题 ❓ 在这里与我们联系。
  • To activate additional products such as Office for macOS, Visual Studio, RDS CALs, and Windows XP, check here.
    要激活其他产品（例如适用于 macOS 的 Office、Visual Studio、RDS CAL 和 Windows XP） ，请点击此处 。
  • To run the scripts in unattended mode, check here.
    要以无人值守模式运行脚本，请检查此处 。
note  笔记
  • The IRM command in PowerShell downloads a script from a specified URL, and the IEX command executes it.
    PowerShell 中的 IRM 命令从指定的 URL 下载脚本，然后 IEX 命令执行该脚本。
  • Always double-check the URL before executing the command and verify the source is trustworthy when manually downloading files.
    在执行命令之前务必仔细检查 URL，并在手动下载文件时验证来源是否可信。
  • Be cautious, as some spread malware disguised as MAS by changing the URL in the IRM command.
    请谨慎操作，因为有些恶意软件会通过更改 IRM 命令中的 URL 来传播伪装成 MAS 的恶意软件。

Features  特征​

  • HWID (Digital License) Method to Permanently Activate Windows
    HWID（数字许可证） 方法永久激活 Windows
  • Ohook Method to Permanently Activate Office
    Ohook 永久激活 Office 的方法
  • TSforge Method to Permanently Activate Windows/ESU/Office
    TSforge 永久激活 Windows/ESU/Office 的方法
  • KMS38 Method to Activate Windows Till the Year 2038
    KMS38 方法激活 Windows 至 2038 年
  • Online KMS Method to Activate Windows/Office For 180 Days (Lifetime With Renewal Task)
    在线 KMS 方法激活 Windows/Office 180 天（终身续订任务）
  • Advanced Activation Troubleshooting
    高级激活故障排除
  • $OEM$ Folders For Preactivation
    用于预激活的 $OEM$ 文件夹
  • Change Windows Edition  更改 Windows 版本
  • Change Office Edition  更改 Office 版本
  • Check Windows/Office Activation Status
    检查 Windows/Office 激活状态
  • Available in All In One and Separate Files Versions
    提供一体化和单独文件版本
  • Fully Open Source and Based on Batch Scripts
    完全开源并基于批处理脚本
  • Fewer Antivirus Detections
    更少的防病毒检测
